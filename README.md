# Ali123 Dropshipping Automation

This repository contains the Ali123 WooCommerce plugin that automates AliExpress Business importing, pricing, and fulfillment workflows.

## Local Development

1. Copy the `ali123` directory into your local WordPress installation's `wp-content/plugins/` folder.
2. Activate **Ali123 Dropshipping Automation** from the WordPress admin.
3. Configure your AliExpress Business credentials under **Settings â†’ Ali123**.

## Feature Highlights

- Import queue management with configurable defaults and pricing rules.
- WooCommerce admin dashboards for automation visibility.
- REST API endpoints for Chrome extension and background workers.
- Background cron integration for bulk imports and tracking sync.

Refer to `docs/ali123_architecture.md` for the full product specification and integration blueprint.
